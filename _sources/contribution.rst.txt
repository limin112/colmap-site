贡献指南
============

我们非常欢迎各种贡献（错误报告、错误修复、改进等），请以在 GitHub 上提交新 issue 和/或 pull request 的形式提交。

请遵循 Google 编码风格指南::

    https://google.github.io/styleguide/cppguide.html

并使用项目提供的 ".clang-format" 文件。

请使用内联文档字符串为函数、方法、类等编写描述 API 的文档，采用以下格式::

    // Short description.
    //
    // Longer description with a few sentences and multiple lines.
    //
    // @param parameter1            Description for parameter 1.
    // @param parameter2            Description for parameter 2.
    //
    // @return                      Description of optional return value.

请为所有新增代码添加单元测试，并确保算法上的“改进”具有普遍适用性，能够在各种数据集上切实提升流程的效果。
